# Photorealism2

## Features

* Physically Based Rendering
* Spectral Rendering
* Realistic Camera Model
* Path Tracing
* Multi Importance Sampling
* OBVH

## Requirements
* CMake(>= 3.12)
* [Intel Open Image Denoise](https://openimagedenoise.github.io/) 
* [Embree(3.6.1)](https://www.embree.org/)

## Build

```zsh
mkdir build
cd build
cmake ..
cmake --build .
```
